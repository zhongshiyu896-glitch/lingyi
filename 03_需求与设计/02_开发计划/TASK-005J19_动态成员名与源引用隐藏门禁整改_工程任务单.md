## 一、任务目标

修复 `check-style-profit-contracts.mjs` 对 runtime mutator 源引用隐藏的剩余绕过，覆盖动态成员名、字符串拼接、`globalThis/window` 多级 element access、`Reflect.get` 和 optional chain。style-profit surface 内凡是指向或可能指向 `Object.defineProperty/Object.defineProperties/Object.assign/Reflect.set/Reflect.apply` 的源引用，必须归一识别；无法静态确认安全时必须 fail closed。

## 二、背景问题

TASK-005J18 已将 runtime mutator “源引用禁用”接入门禁，并关闭函数返回、IIFE、内联字面量、嵌套/包装容器等绕过。但审计官在第 137 份审计中复现以下源引用隐藏绕过：

```ts
Object['define' + 'Property'](item, 'onClick', { value: openHelp })
Object['ass' + 'ign'](item, { onClick: openHelp })
Reflect['s' + 'et'](item, 'onClick', openHelp)

globalThis['Object']['assign'](item, { onClick: openHelp })

Reflect.get(Object, 'defineProperty')(item, 'onClick', { value: openHelp })
Object.defineProperty?.(item, 'onClick', { value: openHelp })
```

这些写法仍是 runtime mutator 源引用或调用，只是通过动态成员名隐藏了 API 名称，必须继续拦截。

## 四、实现要求

### 4.1 动态成员名必须静态折叠

必须识别以下静态可折叠成员名：

```ts
Object['define' + 'Property']
Object[`define${'Property'}`]
Object['ass' + 'ign']
Reflect['s' + 'et']
Reflect[`ap${'ply'}`]
```

要求：

1. 对字符串字面量、模板字面量、字符串拼接表达式做安全静态折叠。
2. 折叠结果命中 mutator 名称时，按源引用禁用规则 fail closed。
3. 折叠失败、包含变量、函数调用或无法确认时，如果对象为 `Object/Reflect/globalThis/window` 或其别名，在 style-profit surface 内 fail closed。

### 4.2 globalThis/window 多级 ElementAccess 必须识别

必须识别以下形式：

```ts
globalThis['Object']['assign'](item, { onClick: openHelp })
window['Object']['defineProperty'](item, 'onClick', { value: openHelp })
globalThis[`Reflect`][`set`](item, 'onClick', openHelp)
window['Reflect']['apply'](Reflect, [item, 'onClick', openHelp])
```

要求：

1. `globalThis['Object']` 与 `window['Object']` 必须等价于 `Object`。
2. `globalThis['Reflect']` 与 `window['Reflect']` 必须等价于 `Reflect`。
3. 多级 element access 必须递归归一化。
4. 任一级 key 无法静态确认安全时，在 style-profit surface 内 fail closed。

### 4.3 Reflect.get 必须识别

必须识别以下形式：

```ts
Reflect.get(Object, 'defineProperty')(item, 'onClick', { value: openHelp })
Reflect.get(Object, 'assign')(item, { onClick: openHelp })
Reflect.get(Reflect, 'set')(item, 'onClick', openHelp)
Reflect.get(globalThis.Object, 'assign')(item, { onClick: openHelp })
Reflect.get(window.Reflect, 'set')(item, 'onClick', openHelp)
```

要求：

1. `Reflect.get(target, key)` 的 target 与 key 均可静态解析时，必须映射为对应源引用。
2. key 支持字符串字面量、模板字面量和字符串拼接折叠。
3. target 为 Object/Reflect/globalThis/window 等价来源且 key 无法静态解析时，在 style-profit surface 内 fail closed。
4. `Reflect['get'](...)` 必须等价处理。
5. `const get = Reflect.get; get(Object, 'defineProperty')` 这类已有别名机制必须继续适配；如无法确认安全，fail closed。

### 4.4 Optional Chain 必须识别

必须识别以下形式：

```ts
Object.defineProperty?.(item, 'onClick', { value: openHelp })
Object?.defineProperty?.(item, 'onClick', { value: openHelp })
Object?.['assign']?.(item, { onClick: openHelp })
Reflect?.['set']?.(item, 'onClick', openHelp)
```

要求：

1. optional property access 与普通 property access 等价处理。
2. optional call 与普通 call 等价处理。
3. optional chain 不能作为绕过 source reference 禁用的理由。

### 4.5 动态成员名 Fail Closed 口径

1. 对 `Object/Reflect/globalThis/window` 及其别名的动态成员访问，如果 key 不能静态证明为安全字段，必须 fail closed。
2. 安全字段白名单必须极小化；不得加入 `defineProperty/defineProperties/assign/set/apply/get` 或任何 action 入口相关 key。
3. 不得通过只读文案、变量名或文件名豁免 runtime mutator 源引用。
4. 不得缩小 TASK-005J13-J18 已关闭范围。
