## 2. 门禁实现摘要
- 新增 runtime codegen 源识别：`Global.eval`、`Global.Function`。
- 将 `eval/Function` 纳入 AST 运行时来源追踪：变量别名、对象解构别名、赋值式解构别名、`globalThis/window` 命名空间访问、静态可折叠字符串成员访问。
- 新增 codegen 专用 source reference 收集器，覆盖直接引用、调用引用、`globalThis/window` 动态成员不可静态解析 fail closed。
- 在 style-profit AST 失败输出中新增独立错误信封：
  - `style-profit forbids runtime code generation entry points; use static readonly helpers（款式利润前端禁止运行时代码生成入口）`
- 保持并复用既有 runtime mutator / dynamic key / explicit action-key / 中文语义门禁，不回退 TASK-005H~TASK-005J19。
